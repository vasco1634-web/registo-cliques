# Registo de Cliques com Data e Hora

Aplicação web desenvolvida no âmbito das disciplinas de ATD e LP.

## Funcionalidades
- Quatro botões para registo de cliques
- Contador sequencial diário (reinicia automaticamente)
- Registo de data e hora de cada clique
- Armazenamento em base de dados SQLite

## Tecnologias
- HTML, CSS e JavaScript
- Python (Flask)
- SQLite
- GitHub
- Replit
